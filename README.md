# 💻 Full-Stack Portfolio Concept (UI/UX Study)

Este projeto é uma **Landing Page de Portfólio** desenvolvida para praticar design de interface, arquitetura de informação e estilização avançada com **HTML5 e CSS3**.

🚀 **Visualize o projeto ao vivo aqui:** [https://jhonatasammuel.github.io/portfolio/](https://jhonatasammuel.github.io/portfolio/)

### 🛠️ Foco do Projeto
O objetivo principal foi a criação de uma interface completa e intuitiva, simulando um ambiente real de contratação de serviços de tecnologia.

### 🌟 Destaques Técnicos:
- **Interface Responsiva**: Estruturação de seções de serviços, portfólio e formulários.
- **Gráficos de Skills**: Implementação visual de níveis de proficiência técnica.
- **Integração de Mídia**: Uso de elementos de vídeo e ícones interativos.
- **Navegação**: Links externos funcionais para redes profissionais (GitHub, LinkedIn e Instagram).

### 📝 Nota sobre o Desenvolvimento
As tecnologias listadas na seção de "Skills" e "Serviços" representam o **planejamento de carreira** e o roteiro de estudos atual. O foco técnico deste repositório está na **estilização (CSS)** e na **estruturação (HTML)**.

### 📈 Evolução Planejada
- Implementação de **JavaScript** para animações de rolagem e validação de formulário.
- Ativação do botão de download de currículo.
